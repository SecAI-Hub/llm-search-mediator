# Support

## Getting help

- **Issues:** [GitHub Issues](https://github.com/SecAI-Hub/llm-search-mediator/issues)

## Before opening an issue

1. Check the [README](README.md) and [examples/](examples/) for usage guidance.
2. Search existing issues to avoid duplicates.
3. Include your Python version, OS, SearXNG URL, and the policy YAML you are using.
